# Vexo Studios Portfolio

A personal portfolio website for **Vexo Studios**, showcasing web design, branding projects, and client services. Built with **HTML, CSS, and JavaScript**, this website is modern, responsive, and designed to highlight professional work for potential clients.

---

## Features

- **Clean & Minimal Design** – Sleek layout with a focus on content and usability.
- **Portfolio Section** – Showcase web and branding projects effectively.
- **Contact Section** – Includes email and Instagram links for easy client communication.
- **Responsive & Mobile-Friendly** – Works seamlessly on desktop, tablet, and mobile devices.
- **Customizable** – Easy to add new projects or update contact info.

---

## Technologies Used

- **HTML5** – Structure of the website.
- **CSS3** – Styling and layout, including modern responsive design.
- **JavaScript** – Interactive elements and enhancements.
